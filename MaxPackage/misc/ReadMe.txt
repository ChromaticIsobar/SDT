This version requires Mac 6.1.7 or higher.

Easy installation (for Max 8.0.2):
- Select **Show Package Manager** in the **File** menu, and browse to install **SoundDesign Toolkit**
- In the **externals** folder, unzip the corresponding OS externals (MacOS, Win32, Win64).
- In the **support** folder, unzip the corresponding OS framework (SDT.framework.MacOSX, libSDT.dll-WIN32, or libSDT.dll-WIN64).

Manual installation:
- Copy the SoundDesignToolkit folder to your **/Applications/Max /packages**.
- In the **externals** folder, unzip the corresponding OS externals (MacOS, Win32, Win64).
 -In the **support** folder, unzip the corresponding OS framework (SDT.framework.MacOSX, libSDT.dll-WIN32, or libSDT.dll-WIN64).


SDT source is available at https://github.com/SkAT-VG/SDT/
